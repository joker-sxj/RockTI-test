# RockTI - 摇滚人格测试

24道趣味选择题，测出你的专属摇滚人格。

**在线测试**: https://joker-sxj.github.io/RockTI-test/

## 16种摇滚人格

经典摇滚 | 硬摇滚 | 重金属 | 朋克 | 后朋克 | 前卫摇滚 | 迷幻摇滚 | 蓝调摇滚 | 民谣摇滚 | 另类摇滚 | 独立摇滚 | 垃圾摇滚 | 英伦摇滚 | 流行朋克 | 盯鞋 | 后摇

## 特色

- 8维摇滚偏好评分模型 (EN/HV/RB/EX/CX/EM/RT/ST)
- MBTI风格详细人格描述（人格画像、超能力、隐藏弱点、最佳拍档、人生BGM等）
- 雷达图可视化
- 混合型人格检测
- 分享海报生成
- 霓虹车库视觉风格
- 移动端优先

## 技术栈

- React 19 + TypeScript
- Vite 7
- Tailwind CSS v4
- Framer Motion
- Recharts
- html2canvas-pro

## 开发

npm install
