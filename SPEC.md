# SPEC.md — RockTI 项目规格说明

## 产品定义

**一句话**：24 道趣味题，测出你的摇滚人格。

**定位**：基于音乐偏好的人格画像测试，不是心理诊断。

**目标用户**：泛摇滚乐迷（18-35 岁，知道几个乐队，不需要专业乐理）

## 核心流程

```
首页 → 开始测试 → 24 道单选题 → 计算 8 维得分 → 匹配 16 种摇滚人格 → 结果页 → 分享海报
```

## 页面结构

| 页面 | 路由 | 核心功能 |
|------|------|---------|
| 首页 | `/` | Logo、说明、开始按钮、16 类型预览 |

| 结果页 | `/result` | 主人格、雷达图、MBTI 风格描述、Top3、分享按钮 |

## 人格描述结构（MBTI 风格）

每种人格包含：

1. **人格画像** — 3-5 句话描述"你是什么样的人"
2. **核心特质标签** — 3-4 个关键词（如 #内向沉浸 #氛围敏感）
3. **超能力** — 1 句话精炼优势
4. **隐藏弱点** — 1 句话温和吐槽
5. **最佳拍档** — 匹配度最高的 2-3 种类型
6. **代表场景** — 画面感强的场景描述
7. **人生 BGM** — 如果你的人生是一首歌
8. **社交签名** — 可直接复制发朋友圈的一句话

## 分享海报规格

- 尺寸：1080 × 1350（Instagram 比例）
- 风格：霓虹海报风（深色背景 + 霓虹色文字）
- 内容：人格名、类型、匹配度、核心标签、签名、测试入口
- 功能：生成图片 + 下载
